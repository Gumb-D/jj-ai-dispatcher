# Codex Task: Repo Cleanup Review

Objective:
Review repository cleanliness and propose cleanup.

Rules:
- Do not delete files.
- Do not push.
- Do not auto-modify tracked files unless explicitly instructed.
- Identify generated files, logs, caches, local runtime files, and machine-specific files.
- Recommend `.gitignore` additions if needed.

Required output:
1. Repo cleanliness summary
2. Files that should be committed
3. Files that should stay local
4. Files that should be ignored
5. Files that may need removal from Git tracking
6. Manual commands only, no execution unless instructed
